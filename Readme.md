# Personal Finanace Tracker
A terminal-based expense tracker I built in C++ as a self project to practice basic C++ and personal finance concepts — all in one single file.

## What it does 
  - Add income and expense records with categories and notes.
  - View all records in a clean table.
  - Filter records by category and see the subtotal.
  - Monthly Report - enter a month and see what u spent.
  - Financial summary showing income, expense, balance, budget used/left and savings rate.
  - Delete every record by ID.
  - Auto saves everything to 'data.txt' and loads it back on startup.

## PF concepts I applied
- **Income vs Expense Tracking** - log where money comes in and where it goes 
- **Category wise spending** - see exactly which category eats most of your budget
- **Net Balance** - total income minus total expenses at a glance 
- **Monthly Budgeting** - filter any month to review your spending 
- **Savings rate** - what percentage of your income u actually kept
- **Budget Used/Left** - how much of your income is already spent 

## C++ Concepts I used 
structs, fixed arrays, loops, switch, fstream for file I/O, setw and setprecision for formatting, function prototypes and input validation — all basic.
